# demo_zhong_wen
Tsinghua No. 70 ChatAI Documentation
Introduction
This project is a chatbot web application built with Streamlit and Google Gemini AI. It helps users learn Chinese vocabulary through conversation.
Main Features
Chat with AI
Save chat history
Load chat history
Clear chat history
How It Works
The user enters a message.
The app sends the message to the Gemini 2.0 Flash model.
The AI generates a response.
The conversation is saved in chat_history.json.
Files
app.py – main application
.env – stores Gemini API key
chat_history.json – stores chat history
Conclusion
This is a simple AI chatbot using Streamlit and Gemini API for learning Chinese.
